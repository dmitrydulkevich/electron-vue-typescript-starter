# electron-vue-typescript-starter

## Project setup
```
yarn install
```

### To start a Development Server
```
yarn electron:serve
```

### To Build Your App
```
yarn electron:build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

### Customize configuration
See [Configuration Reference Vue CLI](https://cli.vuejs.org/config/).
